# Taller-01.-Stakeholders-y-Materialidad-en-el-Sector-Web
**Modulo:** Sostenibilidad Aplicada al Sistema Productivo
**Fecha:** 21 de abril de 2026
**Autor:** Sergio García de Baya

## Que he hecho

He realizado un analisis de stakeholders y temas materiales ASG (Ambiental, Social y Gobernanza) para una consultora de desarrollo web identificando tres grupos de interes:
- Inversores
- Empleados
- Comunidad Local

Tambien he identificado tres temas materiales:
- Gestion de Residuos Circulares (Ambiental)
- Diversidad de Genero en el Liderazgo (Social)
- Etica y Transparencia Corporativa (Gobernanza)

## Archivos del proyecto

- materialidad.xml: Contiene los stakeholders y temas materiales en formato XML.
- materialidad.xsd: Esquema XML que valida la estructura del archivo XML.
